## 기능 설명

### [상품 추가 기능]

- 상품을 입력하여 추가할 수 있습니다.

```
- 입력창에 상품을 입력한 뒤 '추가'버튼 클릭
- 입력창에 상품을 입력한 뒤 키보드 Enter
```

### [상품 수량 증가, 감소]

- 상품의 수량을 증가하거나 감소시킬 수 있습니다.

```
- '+' 버튼을 누르면 수량이 증가합니다.
- '-' 버튼을 누르면 수량이 감소합니다.
```

### [상품 삭제 기능]

- 상품을 삭제시킬 수 있습니다.

```
- 휴지통 버튼을 클릭하면 상품을 삭제할 수 있습니다
- '전체 삭제' 버튼을 클릭하면 상품 전체를 삭제할 수 있습니다.
```
